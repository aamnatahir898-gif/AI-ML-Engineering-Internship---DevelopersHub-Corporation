# AI-ML-Engineering-Internship---DevelopersHub-Corporation
 **Task 1: Exploring and Visualizing a Simple Dataset**
 
 **Task Objective**
The primary goal of this task was to learn the fundamentals of data exploration. This involved loading a real-world dataset, inspecting its structure, performing statistical analysis, and using various visualization techniques to understand data trends and distributions.

**Dataset Information**

**Dataset** 
Iris Dataset

**Source**: Loaded via pandas

**Details**: The dataset consists of 150 samples of iris flowers, with 4 features (Sepal Length, Sepal Width, Petal Length, Petal Width) and 1 target class (Species: Setosa, Versicolor, Virginica).

**Models Applied**
Although the primary focus was visualization, a machine learning model was implemented to validate the data's predictability:

**Algorithm:** Logistic Regression

**Library**: Scikit-learn

**Key Results and Findings**

**Data Integrity:** The dataset was found to be clean with zero missing values.

**Visual Insights:** Scatter plots revealed that the Setosa species is linearly separable from the other two species based on petal dimensions.

**Model Performance:** The Logistic Regression model achieved an impressive Accuracy of ~95.6%.

**Distribution:** Box plots identified minor outliers in the 'Sepal Width' feature, while histograms showed a normal distribution for most features.
